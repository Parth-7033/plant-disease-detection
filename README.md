# 🌿 Plant Disease Detection Using Deep Learning

This project uses a Convolutional Neural Network (CNN) to detect common plant leaf diseases such as **early blight** and **late blight** using image classification techniques. The goal is to assist farmers and agricultural platforms with early disease diagnosis, ultimately improving crop yield and health.

---

## 📌 Features

- Deep learning model (CNN) trained on a dataset of 2000+ labeled plant leaf images
- Detection of 3 classes: **Early Blight**, **Late Blight**, and **Healthy**
- Image preprocessing and data augmentation (rotation, flipping, zoom, etc.)
- Model accuracy: **High generalizability with minimized overfitting**
- Designed to be scalable for real-world use cases in agriculture

---

## 🧠 Tech Stack

- **Language**: Python
- **Libraries**:
  - `TensorFlow / Keras` – for building and training CNN
  - `OpenCV` – for image processing
  - `NumPy`, `Matplotlib`, `Pandas` – for data manipulation and visualization

---

---

## 📊 Results

- Achieved over **90% accuracy** on the validation set
- Overfitting significantly reduced through **early stopping** and **augmentation**
- Model performance is generalizable and robust across unseen leaf samples
